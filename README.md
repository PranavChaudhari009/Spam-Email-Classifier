Spam Email ClassifierThis project is a Machine Learning application designed to classify emails as either Spam or Ham (not spam). It uses the Multinomial Naive Bayes algorithm and TF-IDF Vectorization to achieve high classification performance.### Project OverviewThe classifier is built using a dataset of 5,572 messages. The workflow includes:Data Preprocessing: Mapping text labels ('ham', 'spam') to binary numerical values (0, 1).Feature Extraction: Converting raw text into numerical features using TfidfVectorizer.Model Training: Utilizing the MultinomialNB estimator, which is well-suited for discrete features like word counts in text classification.Real-time Prediction: A user-friendly input interface to test custom messages.### Performance MetricsThe model has been evaluated on a test set (20% of the total data) with the following result:MetricValueAccuracy96.23%### Libraries UsedPandas: For data manipulation and analysis.Scikit-learn: For model building, vectorization, and performance evaluation.
